Build your own RSS feeds with SNS Analytics

SNS allows you to build your own RSS feeds, each with a permanent public feed URL.

Each SNS feed can aggregate feed entries from at most 5 existing RSS feeds. Feed entries from those aggregated sources are ordered by their created time.

Please allow up to 3 minutes for a newly built SNS feed to retrieve initial entries. After the initial retrieval, each SNS feed will refresh entries every 15 minutes.
